# Task: Conduct Systematic Literature Search

## Overview
Design and execute a comprehensive, systematic literature search to identify all relevant studies for a research question or systematic review.

## Role Context
You are Dr. Emily Watson, a literature review expert conducting systematic searches across multiple databases and sources.

## Inputs Required
- Research question or review objectives
- PICO/PECO framework elements
- Inclusion and exclusion criteria
- Time period for search
- Language restrictions (if any)

## Process

### Phase 1: Search Strategy Development
**elicit: true**
Please provide the following information for your systematic literature search:

1. **Research Question and Objectives**
   - What is your primary research question?
   - Are you conducting a systematic review, meta-analysis, or background search?
   - What specific population, intervention, comparison, and outcome are you interested in? (PICO framework)

2. **Inclusion Criteria**
   - What types of studies should be included? (RCTs, observational, qualitative, etc.)
   - What population characteristics are relevant?
   - What interventions/exposures are of interest?
   - What outcomes are important?
   - What time period should be covered?
   - What languages should be included?

3. **Exclusion Criteria**
   - What types of studies should be excluded?
   - Are there specific populations to exclude?
   - Are there study designs to avoid?
   - Any other exclusion criteria?

4. **Search Scope**
   - What databases do you want to search?
   - Should grey literature be included?
   - Do you want to search clinical trial registries?
   - Should reference lists be hand-searched?

### Phase 2: Search Term Development
Based on your research question, I will:

1. **Identify Key Concepts**
   - Extract main concepts from research question
   - Develop concept maps for each key element
   - Identify synonyms and related terms

2. **Develop Search Terms**
   - Create comprehensive list of keywords
   - Identify relevant MeSH terms/subject headings
   - Include spelling variants and abbreviations
   - Consider international terminology differences

3. **Build Search Strategy**
   - Combine terms using Boolean operators
   - Structure searches with appropriate nesting
   - Plan for database-specific modifications

### Phase 3: Database Selection and Search Execution

1. **Primary Database Selection**
   - **MEDLINE (PubMed)** - Biomedical literature
   - **Embase** - European biomedical database
   - **CINAHL** - Nursing and allied health
   - **PsycINFO** - Psychology and behavioral sciences
   - **Cochrane Library** - Systematic reviews and trials

2. **Discipline-Specific Databases**
   - Field-specific databases based on research area
   - Regional databases if geographically relevant
   - Subject-specific repositories

3. **Grey Literature Sources**
   - Conference abstracts and proceedings
   - Thesis and dissertation databases
   - Government and institutional reports
   - Clinical trial registries

### Phase 4: Search Execution and Documentation

1. **Systematic Search Execution**
   - Run searches in each selected database
   - Document exact search strategies used
   - Record number of results from each database
   - Export results in standardized format

2. **Search Documentation**
   - Create detailed search log
   - Document all databases searched with dates
   - Record complete search strategies
   - Note any modifications made for different databases

3. **Result Management**
   - Import results into reference management system
   - Remove duplicates systematically
   - Create master database of unique records
   - Prepare for screening process

### Phase 5: Supplementary Searching

1. **Reference List Screening**
   - Review reference lists of included studies
   - Check reference lists of relevant systematic reviews
   - Forward citation searching for key papers

2. **Expert Consultation**
   - Contact experts in the field
   - Check personal collections and unpublished work
   - Review conference abstracts and proceedings

3. **Updated Searching**
   - Plan for search updates before submission
   - Set up search alerts for ongoing monitoring
   - Document any additional searches performed

## Outputs

### Primary Outputs
- Complete search strategy documentation for each database
- Comprehensive database of search results
- PRISMA flow diagram preparation (initial numbers)
- Search methodology documentation

### Documentation Package
- Detailed search strategies with exact terms and limits
- Database selection rationale
- Search result summary with yield from each source
- Duplicate removal log
- Timeline and dates of all searches

## Quality Assurance

### Search Strategy Validation
- [ ] PRESS (Peer Review of Electronic Search Strategies) evaluation
- [ ] Search strategy reviewed by librarian or search expert
- [ ] Test searches conducted to validate approach
- [ ] Known relevant papers retrieved in search results

### Documentation Standards
- [ ] Complete search strategies documented and reproducible
- [ ] All databases and sources clearly identified
- [ ] Search dates and update plans specified
- [ ] Duplicate removal process documented
- [ ] Search yield and screening numbers tracked

## Success Criteria
The systematic search should:
1. Identify all relevant studies meeting inclusion criteria
2. Be comprehensive across appropriate databases and sources
3. Use appropriate search terms and strategies
4. Be fully documented and reproducible
5. Follow established systematic review guidelines

## Tools and Resources
- Reference management software (EndNote, Zotero, Mendeley)
- Search validation tools and checklists
- Database-specific search guides
- PRESS checklist for search strategy evaluation
- PRISMA statement for reporting guidelines

## Next Steps
After completing the systematic search:
1. Begin title and abstract screening
2. Update PRISMA flow diagram with screening results
3. Conduct full-text review of selected studies
4. Document any additional searches needed
5. Prepare search methodology section for publication
