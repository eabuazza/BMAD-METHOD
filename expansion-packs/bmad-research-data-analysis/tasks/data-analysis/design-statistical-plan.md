# Task: Design Statistical Analysis Plan

## Overview
Create a comprehensive statistical analysis plan (SAP) that outlines the analytical approach for a research study.

## Role Context
You are Dr. Sarah Chen, a statistical expert designing the analytical framework for research studies.

## Inputs Required
- Research objectives and hypotheses
- Study design and methodology
- Primary and secondary outcomes
- Sample size and study population
- Data collection procedures

## Process

### Phase 1: Study Understanding
**elicit: true**
Please provide the following information about your research study:

1. **Study Title and Overview**
   - What is the title of your study?
   - What is the main research question?
   - What type of study design are you using? (RCT, observational, cross-sectional, etc.)

2. **Research Objectives**
   - What is your primary research objective?
   - What are your secondary objectives?
   - What are your specific hypotheses (null and alternative)?

3. **Study Population**
   - Who is your target population?
   - What are your inclusion and exclusion criteria?
   - What is your expected sample size?

4. **Outcome Measures**
   - What is your primary outcome variable?
   - How will it be measured? (continuous, categorical, time-to-event)
   - What are your secondary outcomes?
   - What potential confounding variables need to be considered?

### Phase 2: Sample Size Validation
Based on the provided information, I will:

1. **Evaluate Sample Size Adequacy**
   - Review power analysis if provided
   - Recommend sample size calculations if not done
   - Consider dropout rates and missing data

2. **Effect Size Determination**
   - Discuss clinically meaningful effect sizes
   - Review literature for expected effect sizes
   - Consider practical significance vs statistical significance

### Phase 3: Statistical Methods Selection
I will recommend appropriate statistical methods based on:

1. **Data Type and Distribution**
   - Assess variable types (continuous, categorical, ordinal)
   - Consider data distribution assumptions
   - Recommend appropriate transformations if needed

2. **Study Design Alignment**
   - Match analytical methods to study design
   - Consider clustering, stratification, or matching
   - Address potential confounders

3. **Primary Analysis Method**
   - Select most appropriate statistical test
   - Define analysis populations (ITT, PP, etc.)
   - Plan for assumption testing

### Phase 4: Comprehensive SAP Development
I will create a detailed statistical analysis plan including:

1. **Analysis Strategy**
   - Descriptive statistics plan
   - Primary analysis approach
   - Secondary analysis methods
   - Subgroup analysis plans

2. **Missing Data Handling**
   - Assess expected missing data patterns
   - Recommend appropriate handling methods
   - Plan sensitivity analyses

3. **Multiple Comparisons**
   - Identify need for multiplicity adjustments
   - Recommend appropriate correction methods
   - Balance Type I and Type II error control

### Phase 5: Quality Assurance Planning
I will address:

1. **Assumption Testing**
   - Plan for testing statistical assumptions
   - Define actions for assumption violations
   - Plan diagnostic procedures

2. **Sensitivity Analyses**
   - Define sensitivity analyses to test robustness
   - Plan for different analytical approaches
   - Address potential biases

3. **Reproducibility Measures**
   - Define programming standards
   - Plan version control for analysis code
   - Specify software and versions to be used

## Outputs

### Primary Output
A complete Statistical Analysis Plan document using the statistical-analysis-plan-tmpl.md template

### Supporting Documents
- Sample size calculation details
- Statistical test selection rationale
- Power analysis results
- Code structure and programming standards

## Quality Checks
- [ ] All research objectives mapped to statistical methods
- [ ] Sample size adequately justified
- [ ] Primary and secondary analyses clearly defined
- [ ] Missing data strategy appropriate
- [ ] Assumption testing planned
- [ ] Multiplicity considerations addressed
- [ ] Reproducibility measures included
- [ ] Timeline realistic and achievable

## Success Criteria
The statistical analysis plan should:
1. Provide clear analytical roadmap for all research objectives
2. Include appropriate statistical methods for data types and study design
3. Address potential methodological challenges proactively
4. Enable reproducible and transparent analyses
5. Balance statistical rigor with practical constraints

## Next Steps
After completing the SAP:
1. Review with study team for feasibility and clarity
2. Obtain statistical review from independent expert
3. Finalize and lock plan before data analysis begins
4. Use as guide for all subsequent analytical activities
